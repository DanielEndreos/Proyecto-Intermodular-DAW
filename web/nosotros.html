Sobre nosotros

ElectroHogar es una tienda ficticia dedicada a la venta de electrodomésticos para el hogar. En esta web se muestra una pequeña parte del catálogo, con productos organizados por categorías para que la navegación sea más sencilla.

La idea principal del proyecto es representar de forma simple cómo podría ser la web de una tienda online de electrodomésticos, incluyendo productos, contacto e información general sobre la empresa.

También sirve como base para relacionar esta parte visual con la base de datos del proyecto, donde se almacenan clientes, pedidos, productos, pagos y demás información necesaria para el funcionamiento de la tienda.

Objetivos de la web

  • Mostrar una imagen clara y sencilla de la tienda.
  • Enseñar algunos productos destacados.
  • Permitir al usuario contactar con la empresa.
  • Relacionar la web con la base de datos del proyecto.
